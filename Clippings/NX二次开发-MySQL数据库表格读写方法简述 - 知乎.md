# NX二次开发-MySQL数据库表格读写方法简述 - 知乎
作者：黄盛益 审校：薛剑腾

适用版本：全部

一、概述

在NX二次开发中，有一些数据管理是依靠于MySQL数据库的，特别是一些表格类数据的管理，用户通常都有属于自己的数据库，会要求将这些表格数据都放置在数据库中进行管理，这就要求我们能够对数据库中的表格进行读写操作，其中最常用的数据库为“MySQL数据库”，本文主要介绍如何对MySQL数据库中的表格进行读写操作。
